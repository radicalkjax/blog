Wattz

Wattz is a robotics project aimed at creating a versatile robot capable of performing flips and aerials with moving rotors. The robot is designed to pick up simple objects and navigate rough terrain. This project combines hardware engineering with software development to create a functional and adaptable robot.

The current version (0.01) is the ultra-base version, with plans for more advanced capabilities in future iterations.
