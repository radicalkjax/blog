## running locally

```bash
# clone it
git clone https://github.com/radicalkjax/blog.git
cd blog

# install dependencies
gem install jekyll bundler
bundle install

# run it
bundle exec jekyll serve

# visit http://localhost:4000
```

## tech stuff
